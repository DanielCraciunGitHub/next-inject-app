## Download our Base Template üîΩ

Download our 100% plugin-compatible [base template](/plugins/base-template) with this simple command:

```bash
next-inject init
```

<Accordions>
<Accordion title="What does this command do?">
1. You will receive prompts to select a project name and package manager.
2. Clone our private base template repository and install any dependencies.
3. Initialize a new git repository with `git init`.
4. Run `git commit -m "init"` to make this project yours.

</Accordion>
</Accordions>

Alternatively, you may **bootstrap** an existing Next.js project with Next Inject by running this command:

```bash
next-inject bootstrap
```

<Accordions>
<Accordion title="What does this command do?">
1. You will receive prompts to select a project name.
2. Clone only the **required** files from Github.

</Accordion>
</Accordions>

<Callout title="Warning" type="warn">

Bootstrapped projects naturally suffer from compatibility issues, however we are working to ensure all plugins will work even for bootstrapped projects.

More on bootstrapped projects [here](/plugins/bootstrapped).

</Callout>

  </Step>

## Authenticate with Next Inject üîë

Run this command to authenticate with the CLI:

<Callout title="Warning" type="warn">
  Leaking your API key may lead to security breaches.

You can find your API key in the [dashboard](/dashboard)

```bash
  next-inject auth -k API_KEY
```

</Callout>
